description=[h1]Features[/h1]
description=- Allow players to walk while reading.
description=- Consume any literature when completed.
description=- Completing skillbook will level you up.
description=- Completing skillbook will also give XP Boost.
description=- Don't consume book or magazine that you already completed.
description=- Everything is configurable in the sandbox options.
description=- Should be highly compatible with other literacy mods.
description=
description=[h3]FAQ[/h3]
description=[b]Can I install on existing save-game?[/b]
description=- Yes, for sure!
description=
description=[b]What happen when I read a skillbook?[/b]
description=- You get a level up and XP Boost to help you grind the next level.
description=
description=[b]What happen when I read something I already read in the past?[/b]
description=- Nothing, the character will tell you he already know this book.
description=
description=[b]Why can I read any books without the correct level?[/b]
description=- Make sure the cheat 'Instant Action' is not enabled if you are admin or debug.
